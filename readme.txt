Integrates NextJS frontend with WordPress backend and Groundhogg CRM.

== Description ==

This plugin provides seamless integration between NextJS applications and Groundhogg CRM, offering:

* Bi-directional sync between NextJS and Groundhogg
* Secure API endpoints with authentication
* CORS support for NextJS applications
* Contact management and tracking
* Email integration
* Phone number handling
* Webhook support for real-time updates
* Activity logging
* Rate limiting and caching

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/nextjs-groundhogg-integration`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure the plugin settings under 'NextJS CRM' in the admin menu
